\CWHeader{Лабораторная работа \textnumero 5}

\CWProblem{

Найти в заранее известном тексте поступающие на вход образцы.\newline

\textbf{Формат ввода}\newline
Текст располагается на первой строке, затем, до конца файла, следуют строки с образцами.\newline

\textbf{Формат вывода}\newline 
Для каждого образца, найденного в тексте, нужно распечатать строчку, начинающуюся с последовательного номера этого образца и двоеточия, за которым, через запятую, нужно перечислить номера позиций, где встречается образец в порядке возрастания.\newline
}
